# CMKV

Simulated Annealing with Metropolis-Hastings algorithm for Tetravex Solver

## Launch

Input file format (Each tile on one row in Top/Left/Right/Bottom format):
```
1234
4567
7890
4321
```

Build:
```
make all
```

Launch with input file:
```
./a.out <input_file> <output_file>
```
